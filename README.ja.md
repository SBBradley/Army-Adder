# Army-Adder
このプログラムは軍のリストを生成します。コンソールからユーザーの入力をtxtファイルに書き出します。txtファイルに作ったリストのデータが入っています。

## インスタレーション／使用方法
1.このレポジトリをフォークする。<br>
2.「Code]のドロップダウンからリンクをコピーする。<br>
3.IDEの端末に「git clone ”コピーしたリンク”」を入れる。<br>
4.listBuilder.javaをランする。<br>
